### 3. 创建 `main.py`

```bash
cat <<'EOF' > main.py
from fastapi import FastAPI
from pydantic import BaseModel

app = FastAPI()


class UserInput(BaseModel):
    name: str
    age: int


@app.get("/")
def read_root():
    return {"message": "day2 ready"}


@app.get("/hello")
def say_hello(name: str = "all in ai"):
    return {"message": f"Hello, {name}"}


@app.post("/users")
def create_user(payload: UserInput):
    return {"name": payload.name, "age": payload.age, "status": "created"}
EOF
```

### 4. 启动服务

```bash
uvicorn main:app --reload
```

### 5. 新开终端测试接口

```bash
curl http://127.0.0.1:8000/
curl "http://127.0.0.1:8000/hello?name=Ace"
curl -X POST http://127.0.0.1:8000/users \
  -H "Content-Type: application/json" \
  -d '{"name":"Ace","age":20}'
```
