# 学习记录

这个目录用于记录 `all in ai` 的每日学习过程、打卡内容、完成情况和复盘。

## 记录规则

从现在开始，每天都记录以下内容：

1. 今日学习主题
2. 今日学习任务表格
3. 每项任务对应的“学会标准”
4. 完成情况
5. 结果记录或复盘

## 推荐目录结构

```text
notes/
├── README.md
├── weekly/
│   └── WEEK1.md
├── templates/
│   └── DAILY_CHECKIN_TEMPLATE.md
└── daily/
    └── YYYY-MM-DD.md
```

## 使用方式

- 每天新建一个日期文件
- 按模板填写今日任务
- 学习完成后更新状态和结果记录
- 周末根据每日记录做一次总结
